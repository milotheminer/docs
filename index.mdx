## Authentication

Include your provided API key in the `X-API-Key` header on each lookup request.

```bash
curl "https://api.callheater.com/lookup" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "X-Phone: 4155550123"
```

<Note>
  Lookup data is for informational purposes only. It may be incomplete, delayed, or inaccurate.
</Note>
