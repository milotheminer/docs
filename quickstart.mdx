## Make a Lookup Request

Use `GET /lookup` with a US phone number. Send your provided API key in the `X-API-Key` header.

```bash
curl "https://api.callheater.com/lookup?phone=4155550123&mode=Basic&display=json" \
  -H "X-API-Key: YOUR_API_KEY"
```

The default response is JSON. Set `display=pretty` when you want a plain text response for humans.

```bash
curl "https://api.callheater.com/lookup?phone=4155550123&display=pretty" \
  -H "X-API-Key: YOUR_API_KEY"
```

## Choose a Lookup Mode

Use `mode=Basic` for standard caller context. Use `mode=DeepResearch` when deeper enrichment is available for the number.

```bash
curl "https://api.callheater.com/lookup?phone=4155550123&mode=DeepResearch" \
  -H "X-API-Key: YOUR_API_KEY"
```

## Common Options

Use these query parameters to control the lookup response:

- `phone`: Required phone number to look up.
- `mode`: `Basic` or `DeepResearch`.
- `display`: `json` or `pretty`.
- `realestate`: Set to `true` to include real estate market info in pretty responses.
- `sports`: Set to `true` to include area-code sports team data.

## Response

JSON responses include lookup metadata, phone details, carrier and line type signals when available, caller identity signals when available, and billing metadata for the request.

<Note>
  Lookup data is for informational purposes only. Do not use it as the sole basis for personal, legal, financial, or safety decisions.
</Note>
